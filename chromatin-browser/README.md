# chromatin-browser

A tiny, self-contained genome browser for two chromatin marks over one locus:
H3K27me3 (Polycomb-repressed, broad domains) and Pol II S5p (initiating polymerase,
sharp promoter peaks). Move across the tracks to read signal; called peaks are marked
under the ruler.

All signal is synthetic and generated in-browser - **there are no data files, and it
opens with no server**.

## Run

Open `index.html` in any browser. For a shareable link, enable GitHub Pages on this repo
and point it at `chromatin-browser/`.

A static preview (for this README) is generated by:

```bash
# from the repo root
pip install -r requirements.txt
make chromatin
```

![preview](assets/preview.png)

## What it does

- Synthesizes H3K27me3 as low-frequency broad enrichment and Pol II S5p as Gaussian
  peaks at synthetic promoters (seeded, deterministic).
- Calls peaks per track by threshold + adjacent-bin merge.
- Renders interactive SVG tracks with a **crosshair scrub** (live per-track readout) and
  per-track toggles. Keyboard-accessible; respects reduced motion.

Swap the in-browser generator for a real `bedGraph` to view your own tracks.
